# Personal Portfolio Website

A modern, responsive portfolio website to showcase my projects and skills.

## Features

- Responsive design
- Project showcase
- Skills section
- Contact form
- Dark/Light mode toggle

## Tech Stack

- React
- Next.js
- Tailwindcss
- Framer Motion

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/anilstha1/portfolio-website.git
```

2. Install dependencies:
```bash
cd portfolio-website
npm install
```

3. Start the development server:
```bash
npm start
```
